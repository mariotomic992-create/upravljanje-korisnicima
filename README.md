# Upravljanje korisnicima

Web aplikacija za unos, prikaz i brisanje korisnika.

## Tehnologije
- Linux (Ubuntu)
- Nginx
- MySQL
- PHP
- HTML/CSS

## Funkcije
- Unos novog korisnika (ime, email)
- Prikaz svih korisnika u tabeli
- Brisanje korisnika uz potvrdu
- Prijava na bazu preko forme
